# 📋 小组进度记录

每位成员在提交代码前，先在自己的文件夹里记录完成了什么。

| 文件夹 | 成员 | 职责 |
|--------|------|------|
| `zee/` | Zee | 架构师 + 后端核心 + AI 集成 |
| `zuoyuo/` | 队友 B | 展示型页面（Dashboard、课程、报告、证书） |
| `yyh/` | 队友 C | 交互型页面（设置、笔记、AI 对话、实验、项目） |

## 规则

1. 每次提交前，在自己的文件夹里新建或更新一个 `.md` 文件，记录本次完成了什么
2. 文件命名格式：`YYYY-MM-DD-简述.md`（如 `2026-06-05-项目骨架搭建.md`）
3. 内容包含：完成的功能、修改的文件、遇到的问题（可选）
