### 后端 API（4 个路由文件）
- `server/routes/courses.ts` — 课程 CRUD：
  - GET /api/courses — 获取所有课程列表（含进度计算）
  - GET /api/courses/:id — 获取单个课程详情
  - POST /api/courses — 创建课程（含 AI 异步生成大纲）
  - DELETE /api/courses/:id — 删除课程
- `server/routes/outline.ts` — 大纲 API：
  - GET /api/courses/:courseId/syllabus — 获取大纲（含分段组织）
  - POST /api/courses/:courseId/syllabus/refresh — 重新生成大纲
- `server/routes/report.ts` — 学习报告 API
- `server/routes/certificates.ts` — 证书 API（含已获得/未获得两种状态）
