📖 章节问答 · Python 核心编程 第 7 章 面向对象
🤖
你好！我是你的 AI 学习导师。关于「面向对象编程」这一章，有什么问题我可以帮你解答吗？
👤
能帮我解释一下 Python 中的类和对象的区别吗？
🤖
当然可以！让我们用一个生活中的类比来理解：

类（Class） 就像是一张"建筑设计图"，它定义了建筑应该有什么属性（如面积、楼层数）和功能（如开门、关窗）。

对象（Object） 则是根据这张设计图实际建造出来的"房子"。每个房子都是独立存在的，可以有不同的颜色、家具等具体属性。

在代码层面：

class House:
    def __init__(self, color):
        self.color = color

my_house = House("blue")

这里 House 是类，my_house 就是对象。还有什么疑问吗？😊
